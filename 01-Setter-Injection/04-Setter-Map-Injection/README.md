# Setter Injection with Map

## 📌 Definition

This example demonstrates injecting key-value pairs into a class using Map in Spring.

---

## 📌 Concept

* Student contains a Map of items
* Data is stored as key-value pairs

---

## 📌 XML Configuration

* `<map>` is used for Map injection
* `<entry>` defines key and value

---

## 📌 Java Classes

* Student → contains Map
* Driver → runs the application
* config.xml → defines injection

---

## 📌 Output

1
kula
apple | 1
---------

## jkf | 2

---

## 📌 Key Points

* Used for key-value data
* Supports multiple entries
* Common in real-world applications

---

## 📌 Interview Question

**Q: Difference between List and Map injection?**

**Answer:**
List stores only values, while Map stores key-value pairs.
